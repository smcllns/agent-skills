## The protocol, in one paragraph

Humans engage agents by writing a `#claude`/`#codex`/`#agent` hashtag-directive in any markdown file. The agent does the work, edits the doc body, and wraps the original directive in a callout with a one-line reply. Two protocol states only:

- **`[!NOTE]+`** = active agent thread (scan picks up)
- **`[!DONE]-`** = resolved agent thread (scan skips)

Anything else (`[!NOTE]`, `[!NOTE]-`, `[!DONE]`, `[!DONE]+`) is plain markdown — not for the agent. The marker is the protocol signal; the regex itself does the disambiguation.
